# Phase 005 — Value Proposition Canvas

## Executive Summary
직무/숙련도 세그먼트에 맞춘 가치 제안 5개를 정의하고, 각 세그먼트의 Pain/Gain을 기반으로 메시지·기능 우선순위를 조정했다. "투명한 출처"와 "업계 밈 재작성"의 조합이 모든 세그먼트에서 공통적으로 높은 반응을 얻어 핵심 약속으로 채택되었다.

## Segment Promises
| 세그먼트 | Pain | Gain | NewsKoo 약속 |
| --- | --- | --- | --- |
| 주니어 PM (직무) | 번역체, 문맥 부족 | 빠른 맥락 이해 | 3분 내 읽는 카드 + 원문 CTA + 직무별 요약 |
| 시니어 마케터 | 브랜드 안전성 우려 | 신뢰 가능한 밈/사례 | 투명성 배지와 파트너 샘플 위젯 제공 |
| 운영 큐레이터 | 검수 시간 부족 | 반복 작업 자동화 | 재작성 템플릿과 모델 캐시 제어 |
| 모바일 독자 | 광고 피로, 스크롤 피로 | 가벼운 웃음, 공유성 | 카드형 UX, 원탭 공유, 푸시 개인화 |
| 광고 파트너 | 비용 구조 불투명 | 저비용/로그 기반 딜 | 로컬 AI 비용 공개, Transparency Export |

## Messaging Pillars
1. **믿을 수 있는 출처 표시** — 모든 카드에 모델/검수 로그를 제공.
2. **업계 맞춤 밈** — 직무별 템플릿과 사례를 전면에 배치.
3. **저비용·로컬 AI** — 월 10만원 내 인프라를 강조하여 광고 단가를 낮춘다.
4. **즉시 재활용 가능** — 파트너 위젯·API로 재게시/캠페인에 활용 가능.
5. **모바일 집중 경험** — 카드형 UX와 타임라인 압축.

## Requirements Impact
### Functional
1. **FR-021: 메시지 테스트 슬롯** — 랜딩/앱에 3개의 가치 제안 메시지를 교차 노출.
2. **FR-022: 세그먼트 기반 온보딩** — 직무 선택 시 추천 콘텐츠/알림 기본값을 설정.
3. **FR-023: 공유형 CTA 스택** — 원탭 공유(카카오/Slack/메일)와 Value Prop 배지를 동시 노출.
4. **FR-024: 파트너 증빙 리포트** — 로컬 AI 비용/로그를 주간 PDF로 내보내기.
5. **FR-025: 템플릿 추천 엔진** — 큐레이터 입력(업계/톤)에 따라 템플릿을 제안.

### Non-functional
- **NFR-014 메시지 실험 속도**: 랜딩 테스트 주기 2주, 최소 신뢰도 90%.
- **NFR-015 파트너 리포트 정확도**: 비용/로그 오차 ±5% 이내.
- **NFR-016 온보딩 완료율**: 세그먼트 선택 완료율 80% 이상.

## Dependencies
- FR-021/022는 Phase 090 알림 전략 및 Phase 011 UX 플로우에 전달.
- FR-024는 Phase 018 로그 파이프라인과 Phase 024 파트너 온보딩에서 활용.
- FR-025는 Phase 014 번역 PoC 결과와 Phase 019 워크플로 설계에 연동.

- 워크숍 참석자 16명 중 15명 찬성, 반대 1건은 KPI 연동 세부 수치 필요로 Phase 006에 이관했다.
