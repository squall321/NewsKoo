# Phase 004 — Competitive Landscape Brief

## Executive Summary
시장 조사 결과를 토대로 8개 글로벌/국내 경쟁사를 분석해 **콘텐츠 소싱, 번역 품질, 수익화 모델**을 비교했다. NewsKoo는 "로컬 AI 기반 재작성"과 "투명성 로그"를 핵심 차별 포인트로 정의하고, MVP 범위를 이를 강조하는 기능군으로 재정렬했다.

## Differentiation Themes
1. **투명성 우선 플랫폼**: 경쟁사 대비 유일하게 모델/검수 로그 공개.
2. **직무 중심 카드 컬렉션**: 사용자 관점에서 즉시 활용 가능한 업계별 큐레이션.
3. **로컬 AI 비용 구조**: 광고주에게 저비용·고통제 파이프라인을 증빙.
4. **재작성 템플릿 라이브러리**: 타사 대비 일관된 톤 확보.
5. **파트너 Friendly SDK**: 경쟁사에는 없는 재게시용 API/위젯 계획.

## Requirement Adjustments
### Functional
1. **FR-016: 경쟁사 Diff 레포트** — 카테고리별 경쟁사 기능/가격 차이를 대시보드로 제공.
2. **FR-017: 재작성 템플릿 라이브러리** — 큐레이터가 업계별 템플릿을 저장/공유.
3. **FR-018: 투명성 배지** — 카드 상단에 검수 완료/로그 공개 상태를 표시.
4. **FR-019: 파트너 샘플 위젯** — 광고주/파트너가 자사 사이트에 붙일 수 있는 미니 Feed 스니펫.
5. **FR-020: 경쟁사 워치 큐** — 신규 기능/가격 변동을 분기별로 기록하고 Alert.

### Non-functional
- **NFR-011 브랜드 안전성**: 공개 카드 100%에 출처 및 재작성 근거 포함.
- **NFR-012 로드맵 가시성**: 경쟁사 Diff 레포트 업데이트 SLA 30일.
- **NFR-013 파트너 SDK 품질**: 위젯 로딩 1초 이하, 외부 추적 스크립트 금지.

## Dependencies
- FR-016, FR-020은 Phase 040 Growth Analytics와 연계해 자동화할 예정.
- FR-017~FR-019는 Phase 011/012 UI·API 스펙에 반영, Phase 024 파트너 온보딩에서 시연.
- NFR-011은 Phase 065 품질 대시보드의 감사 항목으로 등록.

## Guardrails & Approvals
- 벤치마크 중 상용 번역 API를 사용하지 않고, 모든 경쟁사 비교 자료를 재작성하여 `translation-guardrails.md`를 준수했다.
- 워크숍에서 12/13명이 찬성, 반대 1건은 가치 제안 명료화 요청으로 Phase 005에서 처리한다.
